# Learn Indian Stock Market

Interactive guide to the Indian stock market — **110+ concepts across 10 chapters**, explained so a 10-year-old can understand.

🌐 **[Live Demo](https://canindya.github.io/learn-indian-stock-market/)**

## What's Inside

| Chapter | Topic | Terms |
|---------|-------|-------|
| 1 | 🏪 What is the Stock Market? | 11 |
| 2 | 💪 Is This Company Any Good? (Fundamentals) | 12 |
| 3 | 📊 Report Card Time (Screening) | 10 |
| 4 | 📈 Reading the Price Chart (Technical Analysis) | 13 |
| 5 | 🏃 Who's Winning the Race? (Momentum) | 9 |
| 6 | 🤖 The Robot's Opinion (Machine Learning) | 8 |
| 7 | 🕵️ Secret Signals (Insider Activity) | 6 |
| 8 | 🏆 The Final Grade (Ranking & Risk) | 9 |
| 9 | 🌤️ Weather Forecast for Markets (Economy) | 11 |
| 10 | 🧰 Your Money Toolkit (Strategies) | 12 |

Each concept includes:
- **Simple explanation** — real-world analogies a kid can understand (lemonade stands, cricket matches, roller coasters)
- **Why it matters** — practical context for how analysts and investors actually use this concept
- **Visual gauge** — mini range indicators for numeric metrics (PE, RSI, scores, etc.)

## Features

- Dark/light theme with persistence
- Collapsible chapters with smooth animations
- Responsive design (desktop, tablet, mobile)
- No framework, no build step, no dependencies
- Zero tracking, zero ads

## Run Locally

Just open `index.html` in a browser. That's it — no server needed.

Or serve it:
```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

## Deploy to GitHub Pages

1. Push to GitHub
2. Go to **Settings → Pages → Source: Deploy from branch → main / root**
3. Your site will be live at `https://<username>.github.io/learn-indian-stock-market/`

## Tech Stack

- Vanilla JavaScript (no framework)
- CSS custom properties for theming
- Google Fonts (Lexend + Syne)
- ~700 lines of JS, ~300 lines of CSS, ~50 lines of HTML
